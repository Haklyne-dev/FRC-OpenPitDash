How to Get API Keys from The Blue Alliance and FRC Nexus

The Blue Alliance

  1. Go to The Blue Alliance website.
  2. Log in or create an account if you don't have one
  3. Navigate to your account settings
  4. Find the section labeled "Read API Keys"
  5. Enter the name for your application (ex. Pit Display), and click "Add New Key"
  6. Copy and paste into the "TBA API Key" field on the settings page

FRC Nexus

  1. Go to the FRC Nexus website.
  2. Navigate to the bottom of the page and click "API"
  3. Log in or create an account if you haven't already
  4. Find the section labeled "Pull"
  5. Click "Create API Key"
  6. Copy and paste into the "Nexus API Key" field on the settings page
